# The Globals Editor

Globals Editor is a program that helps you work with the Globals file for The Hobbit (2003) game. It is written in C++ using ImGui. Makes it easy to work with the Globals File and modify per level logic.

Download link: https://github.com/hobbit-kingdom/globals-editor/releases

## Usage

Get the decompiled Globals File using exporttool.exe and open it in the Globals Editor. instant preview of changes is available inside.

# Main Features

 - **Trigger/Activity types window**: explanation of the different types of triggers/actions.

 - **Modifications**: add/delete/edit triggers, activities and links. For more info follow the youtube tutorial

 - **Merge files**: merge a file with activities/triggers/links with the one you are currently working with. Helps when simillar logic should be added to multiple places.

## How to compile

Use C++ 17 or above (20 preffered) and build with x86 solution platform.
Run .sln to open the project in Visual Studio 2022.
